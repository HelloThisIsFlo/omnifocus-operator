    WRITE_SUFFIXES = ("Command", "Result", "RepoPayload", "RepoResult", "Action", "Spec")

    Helper function `_scan_package_models(package_name: str) -> list[tuple[str, type]]`:
    - Import the package via `importlib.import_module(package_name)`
    - Walk with `pkgutil.walk_packages(pkg.__path__, prefix=f"{package_name}.")`
    - For each module, use `inspect.getmembers(mod, inspect.isclass)` to find BaseModel subclasses
    - Filter: `issubclass(obj, BaseModel) and obj is not BaseModel and obj.__module__ == modname`
    - Skip private classes (name starts with `_`)
    - Return list of (class_name, class) tuples
