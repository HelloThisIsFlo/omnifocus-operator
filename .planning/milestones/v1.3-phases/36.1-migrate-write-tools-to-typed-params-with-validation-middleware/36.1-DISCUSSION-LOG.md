## Middleware Stack Ordering

| Option | Description | Selected |
|--------|-------------|----------|
| Validation INSIDE logging | Logging MW wraps validation MW. Logs show entry, timing, and clean ToolError. Full observability. | ✓ |

**Notes:** None — straightforward pick.
