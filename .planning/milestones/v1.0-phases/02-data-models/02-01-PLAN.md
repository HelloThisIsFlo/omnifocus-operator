From stdlib:

Key ConfigDict parameters:
- alias_generator=to_camel: auto-generates camelCase aliases for all fields
- validate_by_name=True: allows construction with snake_case names
- validate_by_alias=True: allows construction with camelCase names (from bridge JSON)

Key serialization:
- model_dump(by_alias=True): dict with camelCase keys
- model_dump_json(by_alias=True): JSON string with camelCase keys
- model_dump(mode="json", by_alias=True): dict with JSON-compatible types (enums -> strings)
