<objective>
Implement error-serving degraded mode so that fatal startup errors are served as tool responses instead of crashing the MCP server.

Purpose: MCP servers are headless -- crashes are invisible. An agent discovers the error on its first tool call through the normal response channel, with an actionable message saying what went wrong and that the server needs a restart.

Output: ErrorOperatorService class, modified app_lifespan with try/except, simplified __main__.py, full test coverage.
