    1. **RealBridge class** implementing the Bridge protocol via structural typing (do NOT inherit from Bridge protocol -- match the InMemoryBridge pattern):
       - `__init__(self, ipc_dir: Path, *, timeout: float = 10.0)` -- stores ipc_dir, pid (os.getpid()), and timeout
       - `async send_command(operation, params=None) -> dict[str, Any]` -- the main IPC flow:
         a. Generate UUID4 request_id
         b. Build dispatch string: `f"{request_id}::::{operation}"`
         c. Call `await self._write_request(request_id, dispatch)`
         d. Call `self._trigger_omnifocus(dispatch)` (sync, no-op)
         e. Call `await asyncio.wait_for(self._wait_response(request_id), timeout=self._timeout)` wrapped in try/except TimeoutError
         f. On timeout: clean up request file, raise `BridgeTimeoutError(operation=operation, timeout_seconds=self._timeout)` with message "OmniFocus did not respond within {N}s. Is OmniFocus running?"
         g. On success: validate response envelope, clean up both files, return data
       - `_trigger_omnifocus(self, dispatch: str) -> None` -- no-op placeholder (Phase 8 fills in URL scheme trigger)
       - `async _write_request(self, request_id: uuid.UUID, dispatch: str) -> None` -- atomic write:
         a. Filename: `f"{self._pid}_{request_id}.request.json"`
         b. Content: `json.dumps({"dispatch": dispatch})`
         c. Write to `.tmp` first, then `os.replace()` to final path
         d. All inside `asyncio.to_thread()` (single _write closure)
       - `async _wait_response(self, request_id: uuid.UUID) -> dict[str, Any]` -- polling loop:
         a. Response filename: `f"{self._pid}_{request_id}.response.json"`
         b. Poll every 50ms (`asyncio.sleep(0.05)`) checking `response_path.exists`
         c. When found: read and parse JSON
         d. All file ops via `asyncio.to_thread()`
       - `_validate_response(self, raw: dict[str, Any], operation: str) -> dict[str, Any]` -- response validation:
         a. Check `raw` is a dict
         b. If `raw.get("success") is True`: return `raw.get("data", {})`
         c. Otherwise: raise `BridgeProtocolError(operation=operation, detail=f"OmniFocus reported error: {raw.get('error', 'Unknown error')}")`
       - `async _cleanup_request(self, request_id: uuid.UUID) -> None` -- delete request file (missing_ok)
       - `async _cleanup_files(self, request_id: uuid.UUID) -> None` -- delete both request and response files (missing_ok)
       - Helper: `_request_path(self, request_id) -> Path` and `_response_path(self, request_id) -> Path`

    - The timeout error message MUST include "OmniFocus" for user-actionability (per IPC-05)
    - Use `Path.unlink(missing_ok=True)` inside `asyncio.to_thread()` for cleanup
