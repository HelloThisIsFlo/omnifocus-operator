    Add a `model_serializer` to ParentRef that excludes the None branch. Use a `@model_serializer` method that returns a dict with only the non-None key:

    ```python
    @model_serializer
    def _exclude_none_branch(self) -> dict[str, Any]:
        if self.project is not None:
            return {"project": self.project.model_dump(by_alias=True)}
        return {"task": self.task.model_dump(by_alias=True)}  # type: ignore[union-attr]
    ```

    **Fix 1b — Hybrid parent classification (hybrid.py _build_parent_and_project):**

    In `_build_parent_and_project`, before the `if parent_task_id is not None` check, detect when parent_task_id IS the project itself. The project's task_id can be found via `project_info_lookup`: iterate values to find one whose `"id"` matches `parent_task_id`. If found, this is a project parent, not a task parent.

    More precisely: when `parent_task_id is not None` AND `containing_pi is not None`, check if the project_info for this task's containing project has the same task_id as parent_task_id. If so, treat as project parent (fall through to the `elif containing_pi` branch logic).

    Concretely, add a check at the start of the parent resolution block:

    ```python
    # Detect root task in project: parent points to the project's own task row
    is_root_in_project = False
    if parent_task_id is not None and containing_pi is not None:
        info = project_info_lookup.get(containing_pi)
        if info is not None and info["id"] == parent_task_id:
            is_root_in_project = True

    if parent_task_id is not None and not is_root_in_project:
        parent_ref = {"task": {"id": parent_task_id, "name": task_name_lookup.get(parent_task_id, "")}}
    elif containing_pi is not None:
        ...  # existing project branch (unchanged)
    ```

    is_root_in_project = parent_task_id is not None and parent_task_id == project_id
    if parent_task_id is not None and not is_root_in_project:

    elif project_id is not None:
        # Root task in project branch (existing code)
