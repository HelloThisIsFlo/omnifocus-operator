## Milestone Definition of Done

From `.research/updated-spec/MILESTONE-v1.4.md`:

- **Response stripping** — universal stripping of null/empty/falsy across entity responses; `availability` preserved; envelope fields preserved.
- **Effective → inherited** rename across 6 field pairs.
- **Field selection** — `include` (additive semantic groups) and `only` (surgical individual fields) on list tools.
- **True inheritance** — `inherited*` fields on tasks only, populated only when truly inherited from an ancestor, with per-field aggregation semantics.
- **Batch processing** — `add_tasks` / `edit_tasks` accept up to 50 items, best-effort vs fail-fast per tool.
- **Notes graduation** — `actions.note.append` / `actions.note.replace` on edit; top-level `note` removed from edit, retained on add.
- **Count-only** — `limit: 0` returns `{items:[], total:N, hasMore:...}`.
