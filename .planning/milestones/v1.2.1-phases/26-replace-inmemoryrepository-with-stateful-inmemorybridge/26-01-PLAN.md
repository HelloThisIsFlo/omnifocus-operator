Rewrite `tests/doubles/bridge.py` InMemoryBridge class. Preserve BridgeCall dataclass unchanged.

**Constructor changes (per D-01, D-02, D-03):**
- Parse `data` dict into separate mutable lists: `self._tasks = list(seed.get("tasks", []))`, same for `_projects`, `_tags`, `_folders`, `_perspectives`
- Keep existing `self._calls`, `self._error`, `self._wal_path` fields unchanged
- Keep existing `calls`, `call_count` properties unchanged
- Keep existing `set_error()`, `clear_error()` methods unchanged

**send_command rewrite (if/elif dispatch per discretion recommendation):**
- Record BridgeCall first (same as current)
- Check and raise `self._error` if set (same as current)
- Touch WAL path if set (same as current)
- Dispatch based on `operation`:
  - `"get_all"` -> `return self._handle_get_all()`
  - `"add_task"` -> `return self._handle_add_task(params or {})`
  - `"edit_task"` -> `return self._handle_edit_task(params or {})`
  - else -> `return {}`

**_handle_get_all (per D-03):**
- Return `copy.deepcopy({"tasks": self._tasks, "projects": self._projects, "tags": self._tags, "folders": self._folders, "perspectives": self._perspectives})`
- MUST deep copy to prevent `adapt_snapshot()` in BridgeRepository._refresh() from mutating internal state

**_handle_add_task (per D-08, migrated from InMemoryRepository lines 54-93, adapted to dict-level per D-04, D-07):**
- Generate `task_id = f"mem-{uuid4().hex[:8]}"`
- Generate `now = datetime.now(tz=UTC).isoformat()`
- Determine `has_parent = params.get("parent") is not None` (note: payload uses camelCase -- parent comes through as "parent" from model_dump by_alias)
- Build task dict using `make_task_dict()` as template, overlaying with:
  - `id=task_id`, `name=params["name"]`, `url=f"omnifocus:///task/{task_id}"`
  - `added=now`, `modified=now`
  - `note=params.get("note", "")`, `flagged=params.get("flagged", False)`, `effectiveFlagged=params.get("flagged", False)`
  - `inInbox=not has_parent`, `parent=None` (simplified -- bridge doesn't resolve parent refs)
  - `dueDate=params.get("dueDate")`, `deferDate=params.get("deferDate")`, `plannedDate=params.get("plannedDate")`
  - `estimatedMinutes=params.get("estimatedMinutes")`
  - `tags=[]` (bridge doesn't resolve tag names to refs -- tags come as IDs via tagIds)
- Import `make_task_dict` from `tests.conftest` at module level (it's a test file importing from test infrastructure)
- Append task dict to `self._tasks`
- Return `{"id": task_id, "name": params["name"]}`

**_handle_edit_task (per D-09, migrated from InMemoryRepository lines 95-174, adapted to pure dict operations per D-04, D-07):**
- Extract `task_id = params["id"]`
- Find task dict in `self._tasks` by matching `t["id"] == task_id`; raise `ValueError(f"Task not found: {task_id}")` if missing
- Apply simple field updates directly on the dict (no _key_map needed -- both params and stored dicts use camelCase):
  - For each of `name`, `note`, `flagged`, `dueDate`, `deferDate`, `plannedDate`, `estimatedMinutes`: if key present in params, set `task[key] = params[key]`
  - If `flagged` is updated, also set `task["effectiveFlagged"] = params["flagged"]`
- Tag operations (remove then add):
  - `remove_ids = set(params.get("removeTagIds", []))` -- filter `task["tags"]` to exclude matching IDs
  - `add_ids = params.get("addTagIds", [])` -- append `{"id": tid, "name": tid}` for IDs not already present
- Lifecycle:
  - If `params.get("lifecycle") == "complete"`: set `task["availability"] = "completed"`
  - If `params.get("lifecycle") == "drop"`: set `task["availability"] = "dropped"`
- Move (simplified dict-level):
  - If `"moveTo"` in params: extract `container_id = params["moveTo"].get("containerId")`
  - If `container_id is None`: set `task["parent"] = None`, `task["inInbox"] = True`
  - Else: set `task["inInbox"] = False` (parent resolution is simplified at bridge level)
- Return `{"id": task_id, "name": task["name"]}`

**Imports to add:** `copy`, `datetime` (UTC), `uuid4`, `make_task_dict` from `tests.conftest`
