## Decisions Made
- **Private aliases in models/**: Read models (Frequency, OrdinalWeekday) are agent-facing -- they appear in serialized tool output and their JSON Schema must include enum constraints. Private `_FrequencyType` and `_DayName` aliases provide schema correctness without being exported or importable by contracts/.
- **EndByOccurrences gets Annotated**: Despite living in models/, EndByOccurrences is agent-facing (appears in JSON Schema via EndCondition union on contract fields). Adding `Annotated[int, Field(ge=1)]` is acceptable; Plan 02 enforcement test will need to except this case.
- **Patch[Annotated[...]] works**: Pydantic propagates `ge=1` through `Patch[Annotated[int, Field(ge=1)]]`, producing `minimum: 1` in JSON Schema. No outer wrapper needed.
