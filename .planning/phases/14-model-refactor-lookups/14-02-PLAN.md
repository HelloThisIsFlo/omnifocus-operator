    2. **HybridRepository** (`hybrid.py`): Add three new public async methods and three private sync read methods:

       `_read_task(self, task_id: str) -> dict[str, Any] | None`:
       - Open fresh read-only connection
       - Query: `SELECT t.* FROM Task t LEFT JOIN ProjectInfo pi ON t.persistentIdentifier = pi.task WHERE pi.task IS NULL AND t.persistentIdentifier = ?`
       - If no row, return None
       - Build tag lookup for just this task: `SELECT tag FROM TaskToTag WHERE task = ?`
       - Build project_info for parent resolution: query ProjectInfo if row["containingProjectInfo"] is not None
       - Build task_name for parent resolution: if row["parent"] is not None, query `SELECT name FROM Task WHERE persistentIdentifier = ?`
       - Reuse `_map_task_row` with appropriate lookups
       - Return mapped dict

       `_read_project(self, project_id: str) -> dict[str, Any] | None`:
       - Query: same as `_PROJECTS_SQL` + `WHERE t.persistentIdentifier = ?`
       - Build tag lookup for this task ID
       - Reuse `_map_project_row`
       - Return mapped dict or None

       `_read_tag(self, tag_id: str) -> dict[str, Any] | None`:
       - Query: `SELECT * FROM Context WHERE persistentIdentifier = ?`
       - Reuse `_map_tag_row`
       - Return mapped dict or None

       Public async methods follow the same pattern as `get_all`:
       ```python
       async def get_task(self, task_id: str) -> Task | None:
           result = await asyncio.to_thread(self._read_task, task_id)
           if result is None:
               return None
           return Task.model_validate(result)
       ```
       Import Task, Project, Tag from models (use TYPE_CHECKING pattern for ruff).

       Same pattern for get_project and get_tag. Add TYPE_CHECKING imports.

       Same for get_project and get_tag. This is the fallback path (main path is HybridRepository).

       ```python
       @mcp.tool(annotations=ToolAnnotations(readOnlyHint=True, idempotentHint=True))
       async def get_task(id: str, ctx: Context[Any, Any, Any]) -> Task:
           """Look up a single task by its ID. Returns the full Task object."""

           service: OperatorService = ctx.request_context.lifespan_context["service"]

           return result
       ```
