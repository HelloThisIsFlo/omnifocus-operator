    **HybridRepository implementation (GREEN phase):**

    Create `src/omnifocus_operator/repository/hybrid.py`:

    Constructor: `__init__(self, db_path: Path | None = None)`
    - If db_path provided, use it
    - Else check `OMNIFOCUS_SQLITE_PATH` env var
    - Else use default: `~/Library/Group Containers/34YW5XSRB7.com.omnigroup.OmniFocus/com.omnigroup.OmniFocus4/com.omnigroup.OmniFocusModel/OmniFocusDatabase.db`

    `async def get_all(self) -> AllEntities`:
    - Run `_read_all()` via `asyncio.to_thread` (blocking I/O)
    - Return `AllEntities.model_validate(result)`

    `def _read_all(self) -> dict`:
    - Open fresh `sqlite3.connect(f"file:{self._db_path}?mode=ro", uri=True)`
    - Set `conn.row_factory = sqlite3.Row`
    - Query all 5 tables + join table
    - Map rows to dicts matching Pydantic model shape (snake_case field names, since models accept both)
    - Close connection in finally block

    Internal helpers (private module-level functions):
    - `_parse_timestamp(value) -> str | None`: Handle CF epoch float (seconds since 2001-01-01 UTC) and ISO 8601 strings. Return ISO 8601 string with timezone.
    - `_extract_note_text(xml_data: bytes | None) -> str`: Strip XML tags from noteXMLData, return plain text. Empty string for None.
    - `_map_urgency(overdue: int, due_soon: int) -> str`: overdue=1 -> "overdue" (priority), due_soon=1 -> "due_soon", else "none"
    - `_map_task_availability(blocked, date_completed, date_hidden) -> str`: dateHidden -> "dropped", dateCompleted -> "completed", blocked -> "blocked", else "available"
    - `_map_project_availability(effective_status, date_completed, date_hidden) -> str`: dateHidden -> "dropped", effectiveStatus="dropped" -> "dropped", dateCompleted -> "completed", effectiveStatus="inactive" -> "blocked", else "available"
    - `_map_tag_availability(allows_next_action, date_hidden) -> str`: dateHidden -> "dropped", allows_next_action=0 -> "blocked", else "available"
    - `_map_folder_availability(date_hidden) -> str`: dateHidden -> "dropped", else "available"
    - `_build_repetition_rule(row) -> dict | None`: Map SQLite columns to RepetitionRule dict. Return None if repetitionRuleString is NULL.
    - `_parse_review_interval(raw: str | None) -> dict`: Parse "@1w"/"~2m" format into {steps, unit}. Default: {steps: 7, unit: "days"}.

    SQL queries (plain string constants):
    - Tasks: SELECT all needed columns FROM Task WHERE Task does NOT have a matching ProjectInfo row (tasks only, not project root tasks)
    - Projects: SELECT Task columns + ProjectInfo columns FROM Task JOIN ProjectInfo ON Task.persistentIdentifier = ProjectInfo.task
    - Tags: SELECT FROM Context
    - Folders: SELECT FROM Folder
    - Perspectives: SELECT FROM Perspective
    - TaskToTag: SELECT task, context FROM TaskToTag (build a dict[str, list[dict]] mapping task_id -> list of tag refs)

    URL construction: `omnifocus:///task/{id}`, `omnifocus:///project/{id}`, `omnifocus:///tag/{id}`, `omnifocus:///folder/{id}`

    For tag refs in tasks: first query all tags into a lookup dict (id -> name), then query TaskToTag to build tag lists, then attach to tasks during row mapping.

    **IMPORTANT:** The `_read_all` method must be synchronous (called via `asyncio.to_thread`). Only `get_all` is async.

    **IMPORTANT:** For in-memory test databases, the connection URI `file::memory:?mode=ro` won't work (read-only on empty DB). The test helper should create a regular in-memory DB, populate it, then write it to a temp file. Or: use a file-based temp DB for all tests (via `tmp_path`). The HybridRepository constructor accepts a `Path`, so tests pass `tmp_path / "test.db"`.
