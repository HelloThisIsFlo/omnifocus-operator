    **Add `_find_containing_project` method** -- this is CRITICAL for sub-task support.

    In real OmniFocus, `t.containingProject` walks up the task hierarchy to find the
    enclosing project. InMemoryBridge must replicate this by walking the parent chain
    through self._tasks until it finds a task whose parent is a project (ID in
    self._projects), or until it reaches a task with no parent (inbox task).

    The method operates on the SNAPSHOT's task list (already deep-copied, still in
    model-format before conversion). It also needs the project list to check IDs.

    ```python
    def _find_containing_project(
        self, task_id: str, all_tasks: list[dict[str, Any]]
    ) -> str | None:
        """Walk up the parent chain to find the containing project ID.

        Mirrors OmniFocus's t.containingProject behavior: for any task at
        any nesting depth, returns the project ID that contains it.

        Cases:
        1. Task's parent dict has type="task" and parent ID is in self._projects
           -> return that project ID (task is direct child of project root task)
        2. Task's parent dict has type="task" and parent ID is another task
           -> recurse up to that parent task
        3. Task's parent dict has type="project"
           -> return that project ID
        4. Task has no parent (inbox task)
           -> return None

        Uses self._projects to check if an ID is a project.
        Uses all_tasks (the snapshot list) to walk up the chain.
        """
        visited: set[str] = set()
        current_id = task_id

        while current_id and current_id not in visited:
            visited.add(current_id)
            task = next((t for t in all_tasks if t["id"] == current_id), None)
            if task is None:
                return None

            parent_ref = task.get("parent")
            if parent_ref is None:
                # Inbox task -- no containing project
                return None

            if isinstance(parent_ref, dict):
                parent_id = parent_ref.get("id")
                parent_type = parent_ref.get("type", "task")

                if parent_type == "project":
                    return parent_id

                # parent_type == "task": check if parent_id is a project
                is_project = any(p["id"] == parent_id for p in self._projects)
                if is_project:
                    return parent_id

                # Parent is a real task -- walk up
                current_id = parent_id
            else:
                # parent is already a string (shouldn't happen in model format)
                return None

        return None  # Cycle protection
    ```

    **Add `_task_to_raw` method** -- must be an instance method (NOT static) to access
    `self._projects` and `self._find_containing_project`:

    ```python
    def _task_to_raw(self, task: dict[str, Any], all_tasks: list[dict[str, Any]]) -> None:
        """Convert a model-format task dict to raw bridge format in-place.

        Reconstructs the separate parent/project string fields that OmniFocus
        uses in raw format. The project field = containingProject, which is the
        enclosing project regardless of nesting depth.
        """
        # urgency + availability -> status
        urgency = task.pop("urgency", "none")
        availability = task.pop("availability", "available")
        task["status"] = _REVERSE_TASK_STATUS.get(
            (urgency, availability), "Available"
        )

        parent_ref = task.get("parent")
        task_id = task.get("id", "")

        if parent_ref is None:
            # Inbox task: no parent, no project
            task["parent"] = None
            task["project"] = None
        elif isinstance(parent_ref, dict):
            parent_id = parent_ref["id"]
            parent_type = parent_ref.get("type", "task")
            parent_name = parent_ref.get("name", "")

            if parent_type == "project":
                # Model says parent is a project directly.
                # In raw format: parent=None (no parent task), project=project_id
                task["parent"] = None
                task["project"] = parent_id
                task["projectName"] = parent_name
            else:
                # parent_type == "task"
                task["parent"] = parent_id
                task["parentName"] = parent_name

                # Determine the containing project by walking up the chain.
                # This mirrors OmniFocus's t.containingProject behavior.
                is_project = any(p["id"] == parent_id for p in self._projects)
                if is_project:
                    # Direct child of project root task: both parent and project
                    # point to the same ID in raw OmniFocus format
                    task["project"] = parent_id
                else:
                    # Sub-task under another task: walk up to find containing project
                    containing_project = self._find_containing_project(parent_id, all_tasks)
                    task["project"] = containing_project
        else:
            # Defensive: parent is already a string
            task["project"] = None
    ```
