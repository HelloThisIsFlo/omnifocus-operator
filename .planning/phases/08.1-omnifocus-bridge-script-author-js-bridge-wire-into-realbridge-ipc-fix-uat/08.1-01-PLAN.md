OmniFocus globals available in Omni Automation runtime:
- `flattenedTasks` — all tasks
- `flattenedProjects` — all projects
- `flattenedTags` — all tags
- `flattenedFolders` — all folders
- `Perspective.all` — all perspectives
- `Task.Status.*` — task status enum (Available, Blocked, Completed, Dropped, DueSoon, Next, Overdue)
- `FileWrapper.withContents(name, data)` — create file wrapper
- `FileWrapper.WritingOptions.Atomic` — atomic write option
- `URL.fromPath(path, isDirectory)` — path to URL
- `Data.fromString(str)` — string to binary data
- `argument` — global variable set by `arg=` URL parameter
- `console.error(msg)` — log to OmniFocus developer console

IPC Protocol (new — per CONTEXT.md):
- Request file: `{IPC_DIR}/{argument}.request.json` containing `{"operation": "snapshot", "params": {}}`
- Response file: `{IPC_DIR}/{argument}.response.json` containing `{"success": true, "data": {...}}` or `{"success": false, "error": "message"}`
- `__IPC_DIR__` placeholder in bridge.js replaced by Python at load time

Dual-format pattern:
- `if (typeof module === "undefined")` — production IIFE entry
- `if (typeof module !== "undefined")` — test exports via module.exports

```javascript
// Production entry: IIFE
if (typeof module === "undefined") {
    (function() {
        try {
            dispatch(__IPC_DIR__, argument);
        } catch (e) {
            writeResponse(__IPC_DIR__, argument, {
                success: false, error: e.message
            });
            console.error("Bridge error: " + e.message);
        }
    })();
}

// Test exports
if (typeof module !== "undefined") {
    module.exports = { readRequest, writeResponse, handleSnapshot, dispatch, d, pk, rr, ri, ts };

```

Key rules:
- Use `var` declarations, NOT `let`/`const` (OmniFocus JavaScriptCore compatibility)
- Use `function` declarations, NOT arrow functions
- NO `require()`, `import`/`export`, `fs`, `path`, `process` — OmniFocus is NOT Node.js
- All file I/O through `FileWrapper`, `URL`, `Data` OmniFocus APIs
- `__IPC_DIR__` as a var assignment so Python's string replace works cleanly
