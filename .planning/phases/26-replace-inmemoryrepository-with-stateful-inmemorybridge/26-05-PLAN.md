Refactor all ~71 methods in TestEditTask (starting at line ~452) to use the @pytest.mark.snapshot marker and service fixture.

**Mechanical transformation rules** (apply to every method):

1. **Identify snapshot data**: Look at the `InMemoryBridge(data=make_snapshot_dict(...))` call. Extract the kwargs passed to `make_snapshot_dict`.

2. **Default snapshot** (no args to make_snapshot_dict): Remove 3 construction lines, add `service: OperatorService` to method signature. No marker.

3. **Custom snapshot** (has tasks=, tags=, projects= args): Add `@pytest.mark.snapshot(...)` decorator with those kwargs. Remove 3 construction lines, add `service: OperatorService` to method signature.

4. **Needs repo access** (calls `await repo.get_task(...)` or similar after the service call): Also add `repo: BridgeRepository` to method signature.

5. **Needs bridge access** (checks `bridge.calls` or `bridge.call_count`): Also add `bridge: InMemoryBridge` to method signature.

6. **AsyncMock tests** (uses `from unittest.mock import AsyncMock`): Leave completely unchanged — these mock the repository, not using the real bridge stack.

**Example transformations:**

```python
# BEFORE (custom snapshot + repo access):
async def test_patch_name_only(self) -> None:
    from omnifocus_operator.contracts.use_cases.edit_task import EditTaskCommand
    bridge = InMemoryBridge(data=make_snapshot_dict(
        tasks=[make_task_dict(id="task-001", name="Old Name", flagged=True)]
    ))
    repo = BridgeRepository(bridge=bridge, mtime_source=ConstantMtimeSource())
    service = OperatorService(repository=repo)
    result = await service.edit_task(EditTaskCommand(id="task-001", name="New Name"))
    assert result.success is True
    task = await repo.get_task("task-001")
    assert task.flagged is True

# AFTER:
@pytest.mark.snapshot(tasks=[make_task_dict(id="task-001", name="Old Name", flagged=True)])
async def test_patch_name_only(self, service: OperatorService, repo: BridgeRepository) -> None:
    from omnifocus_operator.contracts.use_cases.edit_task import EditTaskCommand
    result = await service.edit_task(EditTaskCommand(id="task-001", name="New Name"))
    assert result.success is True
    task = await repo.get_task("task-001")
    assert task.flagged is True
```
